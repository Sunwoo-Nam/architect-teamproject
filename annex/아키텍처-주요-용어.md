# 아키텍처 주요 용어 설명 (별첨)

> 출처: `[SW Architect 팀과제] On-Device Agentic Platform.docx` — "별첨: 아키텍처 주요 용어 설명" 섹션

- **트랜잭션 원자성(Transaction Atomicity)**: 협상의 모든 단계가 성공하거나, 하나라도 실패하면 아무 일도 일어나지 않은 것처럼 초기화되어 데이터가 꼬이지 않게 보장하는 성질입니다.

- **비동기 협상(Asynchronous)**: 요청을 보낸 뒤 응답이 올 때까지 시스템이 기다리지 않고 다른 작업을 수행하다가, 응답 이벤트가 발생하면 다시 처리를 재개하는 방식입니다. LLM 추론 지연에 대응하기 위해 필수적입니다.

- **RAG(Retrieval-Augmented Generation)**: 외부 지식(웹 정보 등)을 실시간으로 가져와 생성형 AI의 답변 정확도를 높이는 기술입니다.

- **DAG(Directed Acyclic Graph)**: 화살표가 순환하지 않는 그래프 구조로, 작업들 사이의 순서와 의존성을 정의하는 데 최적화된 자료구조입니다.
